## 🚀 Como Usar

1. Clone ou baixe os arquivos
2. Abra `index.html` no seu navegador
3. Personalize com suas informações:
   - Nome da empresa
   - Dados de contato
   - Logo
   - Cores

## 🎯 Customizações Sugeridas

1. **Logo**: Substitua o ícone `<i class="fas fa-code"></i>` por uma imagem
2. **Cores**: Altere as variáveis CSS em `:root`
3. **Conteúdo**: Atualize textos, serviços e informações
4. **Telefone e Email**: Atualize dados de contato
5. **Redes Sociais**: Adicione links corretos

## 📧 Integração de Email

Para funcionar completamente, você pode:
- Usar serviços como Formspree, EmailJS ou similar
- Integrar com um backend PHP/Node.js
- Conectar com Mailchimp ou outras plataformas
